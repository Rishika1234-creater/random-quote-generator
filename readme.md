# Random Quote Generator

A simple and aesthetic **Random Quote Generator** made using  
**HTML + CSS + JavaScript (DOM only)**.

## Features
- Shows a new random quote every time you click the button  
- Clean and modern UI  
- Pure DOM manipulation  
- Beginner-friendly project

## Technologies Used
- HTML  
- CSS  
- JavaScript (DOM)




## How to Run
Just open the `index.html` file in your browser.
